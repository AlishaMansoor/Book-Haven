A full-stack web application that allows users to upload,rate,choose books.

Features:-
User authentication (Sign up / Login / Logout)
Authorization (only owners can edit/delete books & reviews)
Add, edit, delete books
Session-based authentication using Passport.js
MongoDB Atlas cloud database
Cloudinary image upload
Flash messages for user feedback
Responsive UI using Bootstrap

Tech Stack:-
Node.js
Express.js
MongoDB + Mongoose
EJS + EJS-Mate
Passport.js
Bootstrap
Cloudinary
Render (Deployment)

Security
Encrypted passwords
Session-based authentication
Protected routes
